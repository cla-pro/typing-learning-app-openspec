## ADDED Requirements

### Requirement: Safari on iOS (iPad) is a supported browser platform
The system SHALL be usable in Safari on iPadOS (iPad) as a supported browser platform. All pages and interactive controls SHALL render and function correctly in that environment.

#### Scenario: Application loads on Safari iOS
- **WHEN** a user opens the application in Safari on an iPad
- **THEN** the application loads and is fully functional

#### Scenario: Interactive controls render correctly on Safari iOS
- **WHEN** a user views any page with form controls in Safari on an iPad
- **THEN** all controls are visible and operable
